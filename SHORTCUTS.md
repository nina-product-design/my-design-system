# Shortcuts

| Shortcut | What it does |
|----------|-------------|
| **"new project"** | Scaffolds a new page, registers it in main.tsx, adds a dev script on the next available port |
| **"shortcuts"** | Lists all shortcuts |

## Dev Scripts

| Command | What it runs |
|---------|-------------|
| `npm run dev:library` | Design System on :5173 |
| `npm run dev:mobile` | Mobile Page on :5174 |
| `npm run dev:category` | Category Split on :5175 |
| `npm run dev:all` | All of the above |
